This directory contains screenshots and demo GIFs for the Todo App.

### Demo GIFs (GIF format)
- `feature-auth.gif` - Authentication flow demo

## How to Take Screenshots

### iOS Simulator
1. Open your app in iOS Simulator
2. Navigate to the screen you want to capture
3. Press `Cmd + S` or use `Device > Screenshot` from menu
4. Save the file with the appropriate name in this directory

### Creating GIFs
1. Use screen recording tools like:
   - **macOS**: QuickTime Player (File > New Screen Recording)
   - **Third-party**: LICEcap, Kap, or Gifski
2. Record the feature demonstration
3. Convert to GIF and optimize file size (< 2MB recommended)

## Image Guidelines
- **File size**: Keep under 500KB for PNG, under 2MB for GIF
- **Dimensions**: Screenshots should be consistent (iPhone simulator size)
- **Quality**: Use high-quality images that clearly show the features
- **Naming**: Use exact filenames as specified above for proper README display
